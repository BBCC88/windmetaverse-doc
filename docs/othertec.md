## 4EVERLAND

## cognito
使用 Amazon Cognito可以快速轻松地为 Web 和移动应用程序添加用户注册、登录和访问控制功能。详细内容请看[Amazon Cognito官方文档](https://docs.aws.amazon.com/cognito/latest/developerguide/cognito-user-identity-pools.html)
